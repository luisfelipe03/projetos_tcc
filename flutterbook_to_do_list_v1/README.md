# **Proposta do Projeto**

A ideia aqui é criar uma das telas do app [**FlutterBook**](https://github.com/taburity/flutter_book_v3), usando a técnica de **vibe coding**, focando no mínimo de intervenção manual. A tela será uma **ToDo List simples**, permitindo ao usuário criar, visualizar, editar e remover tarefas.

---

# **Objetivo**

Quero gerar a tela completa usando apenas prompts, deixando o fluxo de desenvolvimento o mais automático possível. A meta é ter uma tela funcional, fiel ao design original e com código limpo, seguindo a tradição de boas práticas do Flutter.

---

# **Métricas de Avaliação**

* **Funcionalidade:**
  A tela deve permitir adicionar, visualizar, editar e remover tarefas de forma direta e intuitiva.
* **Qualidade do Código:**
  O código precisa ser organizado, seguindo padrões do Flutter e passando pelas verificações do analisador oficial.
* **Fidelidade ao Design:**
  A interface precisa respeitar o visual já estabelecido no [FlutterBook](https://github.com/taburity/flutter_book_v3) original.
* **Tempo de Desenvolvimento:**
  O processo deve ser rápido, aproveitando ao máximo a automação proporcionada pelos prompts.
* **Esforço de refatoração:**
  O código gerado deve demandar o mínimo de ajustes manuais para atingir os objetivos acima.
