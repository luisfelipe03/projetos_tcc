## Proposta do Projeto

Neste projeto, nosso objetivo é desenvolver uma das telas do app [FlutterBook](https://github.com/taburity/flutter_book_v3) utilizando a tecnica de programação vibe coding. A tela a ser desenvolvida é uma tela simples de ToDo list, onde o usuário pode adicionar, visualizar, editar e remover tarefas. 

## Objetivo
O objetivo principal é desenvolver a tela de ToDo list utilizando a técnica de vibe coding, com o minímo de intervenção humana possível. A ideia é que o código seja gerado automaticamente com base em prompts fornecidos, permitindo uma abordagem mais ágil e eficiente no desenvolvimento da aplicação.

## Metricas de análise
Para avaliar o sucesso do projeto, utilizaremos as seguintes métricas:
- **Funcionalidade**: A tela de ToDo list deve permitir ao usuário adicionar, visualizar, editar e remover tarefas de forma intuitiva e eficiente.
- **Qualidade do Código**: O código gerado deve ser limpo, bem estruturado e seguir as melhores práticas de desenvolvimento em Flutter. Utilizaremos o proprio analisador de código do Flutter para medir a qualidade do código.
- **Fidelidade ao Design**: A interface deve seguir fielmente o design já estabelecido no [FlutterBook](https://github.com/taburity/flutter_book_v3).
